# Report template

```html
<p><strong>yesterday</strong></p>
<ul>
  <li>I worked on <strong>[verified outcome]</strong>.</li>
  <li>I completed <a href="https://example.com/work-item">[verified outcome]</a>.</li>
</ul>
<p><strong>today</strong></p>
<ul>
  <li>I worked on <strong>[verified work already performed today]</strong>.</li>
  <li>I will work on <strong>[concrete next step]</strong>.</li>
</ul>
```

Translate the bullets and headings when local configuration requests another language. When the provider supports HTML, send this through its HTML-content field rather than as escaped text. Prefer outcomes over vague activity. Never invent work for an empty window.
